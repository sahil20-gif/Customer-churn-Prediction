# ANN-Classification-churn-Prediction
- A machine learning web application built with Streamlit to predict whether a customer is likely to churn based on their profile and account details.

![bg jpg](https://github.com/user-attachments/assets/873ee37f-3ab7-4854-9892-54b7456e11ab)



## Features
- Predict customer churn in real time
- User-friendly Streamlit interface
- Preprocessing with saved scaler and encoders
- Machine learning / deep learning model integration
- Fast and interactive predictions

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras

## Project Structure
customer-churn-app/
│
- ├── app.py                 # Streamlit app
- ├── requirements.txt       # Dependencies
- ├── model.h5 / model.pkl   # Trained model
- ├── scaler.pkl             # Scaler
- ├── encoder.pkl            # Encoder(s)
- └── README.md              # Project documentation

## Installation
Clone the repository:
git clone https://github.com/sahil20-gif/Customer-churn-Prediction.git
cd your-repo-name
 ## Install dependencies:
- pip install -r requirements.txt

## Run the app locally:
- streamlit run app.py

## Usage
- Open the app in your browser.
- Enter customer details such as age, balance, credit score, etc.
- Click the prediction button.
- View the churn prediction result instantly.
